    grid

    @function grid($col, $column-max) { ... }

    Description

    Esta función devuelve el porcentaje que ocupa una columna en un máximo de columnas.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $col

    Este parámetro especifica de qué columna queremos averiguar el tamaño

    		Number— none
    $column-max

    Este parámetro especifica cuál es el máximo de columnas

    		Number— none

    Returns

    Percentage —

    Devuelve el porcentaje que ocupa la columna elegida sobre el máximo

    Requires

    • [variable] column-max

    Author

    • Ángel Martínez Gallurt

    no-style-link

    @mixin no-style-link() { ... }

    Description

    Este mixin elimina los estilos por defectos de un enlace o de enlaces dentro del elemento. Para ello, hereda el color de texto que recibiría de no ser por el navegador y elimina el subrayado.

    Parameters

    None.

    Author

    • Ángel Martínez Gallurt

    arrow

    @mixin arrow($rotation: 0deg, $top: auto, $right: auto, $bottom: auto, $left: auto) { ... }

    Description

    Este mixin modifica una flecha rotándola y posicionándola de forma absoluta sobre el contedor.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $rotation

    Indica los grados que rotaremos el elemento

    		Degrees0deg
    $top

    Indica la distancia del elemento con respecto a la parte superior del primer ancestro posicionado

    		Numberauto
    $right

    Indica la distancia del elemento con respecto a la parte derecha del primer ancestro posicionado

    		Numberauto
    $bottom

    Indica la distancia del elemento con respecto a la parte inferior del primer ancestro posicionado

    		Numberauto
    $left

    Indica la distancia del elemento con respecto a la parte izquierda del primer ancestro posicionado

    		Numberauto

    Author

    • Ángel Martínez Gallurt
